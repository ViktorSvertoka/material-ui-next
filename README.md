# Cats & Friends - Next.js + MUI Pet Adoption Landing Page

A modern, responsive landing page built with **Next.js 16**, **Material UI (MUI
v7)**.  
The project showcases a clean UI, adaptive layout, and reusable React components
for a pet adoption platform.

Deployed on **Vercel**.

## Tech Stack

- **Next.js 16**
- **React 19**
- **Material UI (MUI v7)**
